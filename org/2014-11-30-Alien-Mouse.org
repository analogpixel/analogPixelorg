# The pulsating eyes

It may be hard to see from the picture, but the eyes pulsate red. I used an 8pin Arduino Tiny hooked up to a 9V battery and a voltage regulator. The Arduino runs the code to change the voltage to the LEDs by changing the analog voltage out to them.
